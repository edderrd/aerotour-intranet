db::process_conds (  $  conds = null  )  [private]
 

used by other methods to parse the conditions param of a QUERY. If $conds is string then nothing more is done. If it's an array, the first value (index 0) will be consider as the full condition string and all '?' will be replaced by other values in the array (sort of sprintf). You can add a number before a ? to replace it by a given index in the array like 2?

Parameters:
string|array $conds
